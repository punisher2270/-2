работа методов базового и производного классов
Описание процесса
Создаётся базовый класс Animal, у которого есть:

поле name;

метод show_info();

метод speak().

Создаётся производный класс Dog, который наследует Animal.

В нём добавляется новое поле breed.

Используется super().__init__(name) для вызова конструктора базового класса.

Метод speak() в классе Dog переопределяется:

сначала вызывается super().speak();

затем выводится собственное сообщение класса Dog.

Добавлен новый метод wag_tail(), который есть только у производного класса.

В тестовой части создаётся объект:

dog = Dog("Барсик", "овчарка")

после этого вызываются методы:

show_info() — из базового класса;

speak() — переопределённый метод;

wag_tail() — метод производного класса.

Что покажет программа
При запуске будет примерно такой вывод:

python
Это животное: Барсик
Барсик молчит.
Барсик говорит: Гав-гав!
Барсик виляет хвостом. Порода: овчарка
Краткий вывод
Эта программа показывает:

наследование;

использование super();

переопределение метода;

добавление нового метода в производном классе.

class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        print(f"{self.name} молчит.")

    def show_info(self):
        print(f"Это животное: {self.name}")


class Dog(Animal):
    def __init__(self, name, breed):
        super().__init__(name)
        self.breed = breed

    def speak(self):
        super().speak()
        print(f"{self.name} говорит: Гав-гав!")

    def wag_tail(self):
        print(f"{self.name} виляет хвостом. Порода: {self.breed}")


# Тестирование
dog = Dog("Барсик", "овчарка")

dog.show_info()   # метод базового класса
dog.speak()       # переопределённый метод производного класса
dog.wag_tail()    # метод производного класса
